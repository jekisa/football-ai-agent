# AI Football YouTube Agent

## Project Goal

Build a production-ready AI Agent that automatically manages a YouTube channel focused on football.

The system must:

- Collect football news.
- Rank trending topics.
- Generate original YouTube Shorts scripts.
- Create SEO metadata.
- Generate narration.
- Generate subtitles.
- Assemble vertical videos.
- Upload videos to YouTube.
- Analyze channel performance.
- Recommend future content.

## Target Audience

Football fans.

Primary language:

- Indonesian

Secondary language:

- English

## Main Principle

The AI should automate repetitive work while ensuring:

- factual accuracy
- originality
- maintainability
- modular architecture
- production-ready code
